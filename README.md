## 📖 Deskripsi
Aplikasi ini adalah implementasi sistem **pemesanan tiket bioskop online sederhana** berbasis **Java JSP** yang **tidak menggunakan database**. Pengguna dapat:

🎬 Menjelajahi daftar film yang tersedia  
🕒 Memilih jam tayang dan jumlah tiket  
🧾 Mendapatkan struk pemesanan lengkap dengan total harga  

Semua data disimpan **sementara menggunakan session**, menjadikannya cocok sebagai latihan UTS mata kuliah **Java Lanjutan**.

---

## 🚀 Fitur Utama

✅ Menampilkan daftar film lengkap:  
 • Judul  
 • Genre  
 • Durasi  
 • Harga tiket  
 • Jam tayang  

📝 Formulir pemesanan interaktif  
🧾 Struk digital dengan perhitungan total harga otomatis  
💻 Desain **responsif** menggunakan **Bootstrap 5**  
📦 Penyimpanan sementara menggunakan **session & array**

⚙️ Cara Menjalankan
Prasyarat
Java Development Kit (JDK) 8 atau lebih tinggi

Apache Tomcat 8.5 atau lebih tinggi

Browser modern (Chrome, Firefox, Edge)

Pastikan JDK dan Tomcat sudah terinstal

Copy folder project ke direktori webapps Tomcat, misal:
C:\apache-tomcat\webapps\UTS_JSP_22110233_SutantoDwiPutra

Jalankan Tomcat:

Buka Command Prompt

Masuk ke folder bin Tomcat:
cd C:\apache-tomcat\bin

Buka browser dan akses:

http://localhost:8080/UTS_JSP_22110233_SutantoDwiPutra

Untuk menghentikan server, jalankan:
shutdown.bat
